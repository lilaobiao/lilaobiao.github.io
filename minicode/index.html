# 小程序概述

小程序是一套运行在微信环境里的代码，表面上看就跟通过微信浏览器打开网页差不多。但小程序并非完全的web页面，而是原生组件和html元素的结合体。小程序的页面渲染也和普通的html不同，普通的html运行在浏览器中，渲染时是单线程的，而小程序的渲染是双线程的，即逻辑代码（js）和视图代码(wxml)分别在不同的线程执行。

之所以叫小程序，是因为它依赖于微信宿主环境，在没有做分包处理的情况下，打开一个小程序等同于它的代码全部下载到了手机里。所以小程序有很多限制，不能用于做大型项目开发，也不适合业务太过复杂的项目。一般的小程序，页面都介于10-20页之间。
